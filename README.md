<h1>一个基于Django开发的博客系统:</h1>

======================


>运行环境：Python2.7 和 Django1.11


======================

<h4>现有功能

- xadmin后台管理
- markdown编辑器
- Bootstrap前端模板
- RSS订阅
- 标签分类与日期归档
- MPTT树形结构展示文章评论
- haystack文章内容搜索，高亮显示
- 数据库使用db.sqlite3

======================


<h4>如何运行</h4>

>git clone https://github.com/styleolder/blog.git

>python manage.py runserver

>访问http://127.0.0.1:8000/blog

>默认用户:admin/admin123456
